# Voice Check Supplement: Vex Whitepaper

FINRA 2210 retail communication standards apply to all content in this repository.

## Banned promotional language

```voice-check-phrases
exciting opportunity
massive opportunity
poised for growth
well-positioned to dominate
next big thing
game-changing
best-in-class
unparalleled
risk-free
safe investment
```

## Banned promotional words

```voice-check-words
revolutionary
transformative
groundbreaking
disruptive
guaranteed
```

## Banned prediction patterns

```voice-check-regex
\bwill\s+(likely\s+)?IPO\b
\bexpect(ed)?\s+to\s+(grow|increase|rise|appreciate|outperform)\b
\bguaranteed\s+(returns?|liquidity|exit)\b
\bprojected\s+(returns?|growth|performance)\b
```
